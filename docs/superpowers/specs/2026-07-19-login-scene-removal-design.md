# 로그인 WebGL 장식 제거 설계

## 목표

로그인 페이지 배경 위에 표시되는 WebGL 폴리 장식만 제거한다.

제거 대상:

- 회색 폴리 구 4개
- 폴리 나무 2개
- 떠 있는 폴리 책 1개

유지 대상:

- 사용자가 제공한 배경 GIF
- 글래스모피즘 로그인 카드와 카드 내부 책 아이콘
- 우측 상단 학교 표시
- 하단 저작권 문구

## 구현 방향

`src/app/login/page.tsx`의 `LowPolyBackdrop`에서 로그인 페이지에 전달하는
`scene` 속성을 비활성화한다. 이 변경은 로그인 페이지의 WebGL 캔버스와 장식
장면 생성을 함께 중지한다.

공용 `LowPolyBackdrop`, 학생·교사 페이지 장면 생성 코드와 기존 폴리 자산은
수정하지 않는다. 따라서 다른 페이지의 장면 동작에는 영향을 주지 않는다.

## 검증

- 로그인 페이지 렌더링 테스트에서 장면 비활성화 속성을 확인한다.
- 로그인 카드, 학교 표시, 저작권 문구가 계속 렌더링되는지 확인한다.
- 배경 GIF와 글래스 카드 스타일 계약 테스트를 유지한다.
- 전체 테스트, ESLint, 프로덕션 빌드를 실행한다.
- 배포 후 운영 로그인 페이지에서 장식 캔버스가 없고 GIF·로그인 UI가 정상인지 확인한다.
